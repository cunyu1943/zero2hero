## 📢 关于

💯 本仓库主要面向 Java、C/C++ 技术方向学习者、校招求职者的一些基础知识总结，包括操作系统、计算机网络、数据结构、算法与刷题、数据库、语言、框架、程序库等知识以及面/笔试经验等信息，记录自己的技术生涯，希望我们都能从菜鸟进阶为大神！
